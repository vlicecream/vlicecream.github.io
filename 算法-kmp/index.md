# kmp算法


## ***什么是kmp***

1. *wiki百科*

   *Knuth-Morris-Pratt字符串查找算法，简称为KMP算法*

2. *作用*

   - *可在一个字符串`s`内查找一个词的出现位置。一个词在不匹配时本身就包含足够的信息来确定下一个匹配可能的开始位置，此算法利用这一特性以避免重新检查先前配对的字符*
   - *他可以在O(n + m)的时间复杂度内实现两个字符串的匹配*
