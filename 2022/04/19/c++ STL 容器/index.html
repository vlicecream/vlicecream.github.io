Array

/*
	namespace jj01
	
	数组是固定大小的序列容器, 里面的元素是一段连续的内存地址
	通用格式 array<类型名, 元素个数>数组名;
		ps: 元素个数不能是变量 因为长度固定
	array并不支持(也就是不允许你指定)分配器的哦～
*/

Vector

/*
	namespace jj02
	
	vector 是只能往后扩展 如果空间不够 扩容规则是两倍增长的速度
	这个两倍增长其实是换了个内存地址 虽然将原来的元素一点一点移过去 所以这个过程相比是很缓慢的
	如果vector的元素类型是int，默认初始化为0
	如果vector元素类型为string，则默认初始化为空字符串
*/

List

/*
  namespace jj03
  
  扩容是不可能在原来的地方也就是内存地址扩容两倍 所以扩容后内存地址肯定会变 而且扩容的很缓慢 
  	但是list不一样 他是有指针的 相当于一个萝卜一个坑～ 每放一个元素进去 就会有指向这个元素的地址
  list 不可以用全局的 ::sort 方法 必须要用他自己类自带的sort算法 因为分配器的原因
*
