c++ 头文件-类声明

开文说明

本文章大多数都是根据以下代码为模范讲解

class complex {
public:
  complex(double r = 0, double i = 0) : re(r), im(i) {}
  complex &operator+=(const complex &);
  double real() const { return re; }
  double imag() const { return im; }

private:
  double re, im;

  friend complex &__doapl(complex *, const complex &);
};

Class 是由 class head & class body 组成的

inline函数

函数若在class body内定义完成 成为inline候选

但是不是所有的函数都能变成inline

简单点讲 函数复杂就不会变成inline 终究还是编译器去决定

Access level访问级别

public: 公开的

private: 私有的

还有一种 保护的 先不讲~

Example:

class complex {
public:  // 公有
  complex(double r = 0, double i = 0) : re(r), im(i) {}
  complex &operator+=(const complex &);
  double real() const { return re; }
  double imag() const { return im; }

private:  // 私有
  double re, im;

  friend complex &__doapl(complex *, const complex &);
};

{
  complex c1(2, 1);
  cout << c1.re;
  cout << c1.im;
}  // 错误

{
  complex c2(2, 1);
  cout << c2.real;
  cout << c2.imag;
}  // 正确

上述错误的方法 创建了一个对象然后 直接拿 是不可以的 因为是private 这些数据必须通过自己的兄弟传递出去
