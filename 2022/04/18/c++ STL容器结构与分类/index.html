c++ STL 容器结构与分类

容器-结构与分类

/*
Sequence Containers 循序式容器 
	数组Array: 一段连续的内存空间
  Vector: 前面不能动 如果放元素 后面会自动扩充  自然就是分配器所分配内存空间
  双端队列Deque: 两端可进可出
  列表List: 并不是连续的 每个元素都是指针双向连接起来的
  单向列表Forward-List: 指针单向连接起来的
*/ 

tip:
	python中的list可不是c++这种的list cpython中的是基于数组实现的顺序表
  c++ 标准库中的list是双向链表 拥有双向链表的优点
/*
Associative Containers 关联式容器(key-value)
	集合 Set/Multiset:  key即是value value则是key
		Set元素不重复 Multiset元素可以重复
	map/Multimap: key-value
		map key不重复 Multimap key可重复
	在标准库里面并没有必须规定内部实现用什么数据结构实现
	这两者在c++内部可用红黑树实现 高度平衡二叉树
	golang里map则用hashTable实现
*/ 
/*
Unordered Containers 也是一种关联式容器 不定序元素容器(里面的元素没有顺序)
	HashTable Separate Chaining
		在golang里详细介绍了hashTable
		主要内容有取模法，哈希冲突，渐进式扩容
*/
