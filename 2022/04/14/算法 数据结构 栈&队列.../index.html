栈

栈

​ What?

​ 栈就是先进后出

​ Example

​ 一个箱子 你放4本书进去 拿书的时候只能从上面先拿 这就是先进后出

​ time complexity

​ select O(n)

​ insert O(1)

​ remove O(1)

队列

​ What?

​ 队列就是先进先出

​ Example

​ 排队的时候先来的人就可以先出去 这就是先进先出

​ time complexity

​ select O(n)

​ insert O(1)

​ remove O(1)

双端队列

​ What?

​ 就是一个队列 可以理解为stack和queue的结合体

​ 可以往最前面添加元素 or 删除元素

​ 也可以往最后面添加元素 or 删除元素

​ time complexity

​ select O(n)

​ append O(1)

​ remove O(1)

优先队列

​ what?

​ 元素具有优先级

​ example

​ 好比你是vip 所以就可以优先获取资源

​ time complexity

​ 插入操作 O(1)

​ 取出操作 O(logn) ps: 按照元素的优先级取出

​ 优先队列的底层具体实现

​ 较为多样和复杂 可以用heap bst treap等
