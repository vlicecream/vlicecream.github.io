python 入门知识点

一、变量

1.1 什么是变量

变量 正如他的名字 分成两个字来看  变和量   量是个意思很广泛的词，就是指事物的状态，然后再看变，变化的意思，合起来看就是一个变化的事物的状态

1.2 为什么要有变量

因为在三次元人类世界中，存在很多变化的事物，人类要想跟计算机沟通，让计算机替人类办事，首先就得在计算机中弄一个人类世界，自然也就要有一个计算机能理解的变化的事物

1.3 怎么使用变量（先定义 后调用）

1.3.1 变量的定义与使用

# 定义其实就是   “变量名”    =     “值”  
name = "Ame"
age = 18
height = 1.8

1.3.2 变量名的命名规范

# 命名规范
1. 变量名只能是 字母、数字或下划线的任意组合
2. 变量名的第一个字符不能是数字
3. 关键字不能声明为变量名，常用关键字如下
['and', 'as', 'assert', 'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except', 'exec', 'finally', 'for', 'from','global', 'if', 'import', 'in', 'is', 'lambda', 'not', 'or', 'pass', 'print', 'raise', 'return', 'try', 'while', 'with', 'yield']

1.3.3 变量名的命名风格

# 风格一：驼峰体
AgeOfTony = 56 
NumberOfStudents = 80
# 风格二：纯小写下划线(在python中，变量名的命名推荐使用该风格)
age_of_tony = 56 
number_of_students = 80

1.3.4 变量值的三大特性

变量名 id   
值的类型 tybe	
变量值   value

三、 常见的数据类型

3.1 数字

3.1.1 整数INT

整数（int） 定义：
	age = 18
	print(type(age))
	int
# 一般用来 记录 年龄 等级 qq 等等

3.1.2 浮点型 FLOAT

浮点型（float） 定义：
	height = 1.8
# 一般用来 记录 身高 体重  薪资 等等

3.2 字符串

在python中，加了引号的字符就是字符串’
	定义：name = “Ame”
# 一般用来记录 姓名 性别  种族等等
数字可以进行加减乘除等运算，字符串呢？让我大声告诉你，也能
是的，但只能进行"相加"和"相乘"运算
注意1：字符串相加的效率不高
字符串1 + 字符串3，并不会在字符串1的基础上加字符串2，而是申请一个全新的内存空间存入字符串1和字符串3，相当字符串1与字符串3的空间被复制了一次，

注意2：只能字符串加字符串，不能字符串加其他类型

字符串拼接（只能在字符串之间进行，且只能相加或相乘）

3.3 列表

列表则是中括号[]内，其中以逗号分隔，里面有n个数据。

	list = [12, 15, 56]

列表嵌套:

	list = [12, 15, 56,[58, 46]]

3.4 字典

	在{}内用逗号分隔，可以存放多个key:value的值，value可以是任意类型
定义:
	info={'name':'egon ','age':18,'sex':18}

	用于标识：存储多个值的情况，每个值都有唯一一个对应的key，可以更为方便高效地取

字典嵌套：info={
    'name':'egon',
    'hobbies':['play','sleep'],
    'company_info':{
        'name':'Oldboy',
        'type':'education',
        'emp_num':40,
    }
}
print(info['company_info']['name'])

3.5 布尔

bool值只有 True 和 False

	a = 3

	b = 5

如果a > b 则False    如果a < b True

布尔类型的重点知识！！！：

#所有数据类型都自带布尔值
1、None，0，空（空字符串，空列表，空字典等）三种情况下布尔值为False
2、其余均为真

3.6 重点：

#1.可变类型：在id不变的情况下，value可以变，则称为可变类型，如列表，字典

#2. 不可变类型：value一旦改变，id也改变，则称为不可变类型（id变，意味着创建了新的内存空间）

6.6 解压赋值

name_list = ['jason','egon','tony','kevin']
   name1 = name_list[0]
   name2 = name_list[1]
   name3 = name_list[2]
   name4 = name_list[3]
   print(name1,name2,name3,name4)
   # 解压赋值
   name1, name2, name3, name4 = name_list  # 按照位置一一对应赋值
   print(name1, name2, name3, name4)

  # name1, name2, name3 = name_list  # 少了不行
  # name1, name2, name3, name4, name5 = name_list  										# 多了不行

  # 可以打破个数必须相等的情况
  l = [11, 22, 33, 44, 55, 66, 77, 88, 99]
  a, b, *_, c = l
  print(a, b, _, c)
  a, b, *_ = l
  print(a, b, _)
