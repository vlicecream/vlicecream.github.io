python 基本数据类型

一、基本数据类型的内置方法

1.1 整数和浮点数

1、整数
# 整数的转换
    age = "18"
    print = int(age)
# 非整数型的是转换不了整数，比如
	height = 1.8
    print = int(height)  # 这个是错误的，带了小数点，应该转换成浮点型
    name = "Ame"
    print = int(name)  # 这个是错误的，因为字符串是英文字母组成的
	print(int(11.11))  # 11
    print(int("11.11"))  #报错 因为里面是字符串型的浮点型 
2、浮点型
# 浮点型的装换
	height = 180.1
    print(float(180.1))  # 180.1
    print(float("180.1"))  # 180.1
    print(float(180))  # 180.0
    print(int(11.11))  # 11

1.2 字符串

1、字符串的查找(索引)   索引是从0开始的
	name = "ameameame"
    print(name[0])  # a
    print(name[-1])  # e
2、字符串的查找(切片)
	name = "ameameame"
    print(name[0:6:1])  # ameame 1 乃是步长的意思
    print(name[0:6:2])  # aem	2 乃是步长的意思
    print(name[-1:-6:-1])  #emaem 	3 因为查找默认是从左到右，所以得加-1来从右到左的查找
3、字符串的长度
	name = "ameameame"
	print(len(name))  # len是能弄清楚字符串的长度  name长度为9  提示 空格也算字符
4、字符串的.sprit()用法
	# .strip是去除字符串指定首尾的字符，默认不指定的时候则是去除首尾空格
    name = "      ameameame    "
    print(name.strip())  # ameameame 去除了首尾空格
    name = "%%ameameame%%"
    print(name.strip("%"))  # ameameame 去除了首尾%
    name = "%%ameame%a%m%eame%%"
    print(name.strip("%"))  # ameame%a%m%eame  只能去首尾，中间的不能去除
    # 如何才能去除左边的保留右边的(.lstrip)
    name = "%%ameameame%%"
    print(name.lstrip("%"))  # ameameame%%
    # 如何才能去除右边的保留左边的(.rstrip)
    name = "%%ameameame%%"
    print(name.rstrip("%"))  # %%ameameame
5、字符串的判断在不在字符串里面(in      not in)
	name = "ameameame"

6、字符串的.split()用法
	str = "1!2!3!4!5!6!7!8!9"
    list = str.split("!")
    print(list)  # ['1', '2', '3', '4', '5', '6', '7', '8', '9']
    # 因此可以看出字符串转换成了列表，切割的"！"也不会被保留，所以切记，以后碰到字符串转列表就可以用.split用法。
    list1 = str.split("!",maxsplit=2) # print(list)  # ['1', '2', '3!4!5!6!7!8!9']
    # 默认.split是从左到右切割
    list2 = str.rsplit("!",maxsplit=2)  # ['1!2!3!4!5!6!7', '8', '9']

1.6 集合

1.什么是集合
	大括号括起来 内部可以存放多个元素
	元素与元素之间逗号隔开  元素只能是不可变类型(整型 浮点型 字符串 元组)
    
2.集合的应用场景
	只有两种情况才会考虑使用集合
        1.关系运算
            两个群体之间判断 交叉并集
                共同好友 共同关注 共同点赞...
        2.去重操作
            去除重复的元素
            
3. 集合的定义
	# 因为跟字典很像，所以定义空{}的时候 默认是空字典 所以需要加set来声明这是集合
    
4. 集合的去重操作
	# 去重不保留顺序
	l = [11,22,33,22,22,33,11,22,55,66,66,77,77]
    set = set()
    for i in l:
        if i not in set:
            set.add(i)
    print(set)
	
    
    # 去重保留顺序
	l = [11,22,33,22,22,33,11,22,55,66,66,77,77]
    new_l = []
    for i in l:
        if i not in new_l:
            new_l.append(i)
    print(new_l)

5. 关系运算
	f1 = {'jason','egon','kevin','tony'}
	f2 = {'jason','jack','oscar','kevin'}
I 共同好友
	f1 & f2
II f1独有的好友
	f1 - f2
III f2独有的好友
	f2 - f1
IV 所有的好友
	f1 | f2
V  各自的好友 不包括共同好友
	f1 ^ f2
6. 父集 子集
"""
父级:A包含B 那么A就是B的父级
子级:A包含B 那么B就是A的子级
"""
s1 = {1,2,3,4,5}
s2 = {1,2,3}
print(s1 > s2)  # True
print(s1 < s2)  # False

二、可变类型与不可变类型

1.1 可变类型

"""
可变类型
    值改变 内存地址不变
"""
# 可变类型
l1 = [11,22]
print(id(l1))  # 2133952624128
l1.append(666)
print(id(l1))  # 2133952624128

1.2 不可变类型

"""
不可变类型
    值改变 内存地址一定变
"""
# 不可变类型
    a = 1
    print(id(a))  
    a = 2
    print(id(a))
