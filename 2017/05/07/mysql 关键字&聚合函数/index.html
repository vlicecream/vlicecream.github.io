mysql 关键字 聚合函数

一、ROUND

1.什么是round
	ROUND(f,p) 返回 f 舍入到 p 位小数。
	小数位数可能是负数，这将四舍五入到最接近的 10（当 p 为 -1）或 100（当 p 为 -2）或 1000（当 p 为 -3）等。
	可以保留小数并四舍五入
        round(1231.86,0)  # 1232
        round(1231.86,1)  # 1231.9
        round(1231.86,-3)  # 1000
2.语句演示
	select name, round(population/1000,1) from xxx;

二、CASE

1.什么是case
	CASE 允许您在不同条件下返回不同的值。
	如果没有任何条件匹配（并且没有 ELSE），则返回 NULL。
	
2.语句演示
	case when 条件1 then 值1
		when 条件2 then 值2
		else def_valve end
		
		
	select name, population,
	case when population<1000000 then 'small'
         when population<10000000 when 'medium'
         else 'large' end
    from world

三、ORDER BY

1. 什么是order by
	能对数据进行升序降序排队

2. 语句演示
	select name, continent from world where name in ('Argentina', 'Australia') order by name asc;  # ASC 升序
	select name, continent from world where name in ('Argentina', 'Australia') order by name desc;  # DESC 降序

四、CONCAT

1. 什么是concat
	可以拼接两个或者多个字符串
	这个操作是串联

2. 语句演示
	select name, concat(population,'%') from world;

五、SUM

1. 什么是sum
	求和

2. 代码演示
	select sum(gdp) from world where continent = 'Africa';

六、 MAX&MIN

1. 什么是max&min
	比较最大&比较最小

2. 代码演示
 	select post,max(salary) from emp group by post;

	select post,min(salary) from emp group by post;

七、DISTINCT

1. 什么是distinct
	去重
	一定要注意 必须是完全一样的数据才可以去重！！！
	一定不要将逐渐忽视了 有逐渐存在的情况下 是不可能去重的

2. 代码演示
	select distinct id,age from emp;

八、 COUNT

1. 什么是count
	可以总计次数

2. 代码演示
	select continent, count(name) from world group by continent

九、GROUP BY

1. 什么是group by
	可以进行分组
	简单来说：1、与distinct类似，自带去重效果
    配合聚合函数使用{sum()、max()、min()、avg()、count()}
    配合HAVING

2. 代码演示
	select continent, name from world group by continent

十、 HAVING

1. 什么是having
	"""
    having的语法根where是一致的
    只不过having是在分组之后进行的过滤操作
    即having是可以直接使用聚合函数的
    """
    
2. 代码演示
	select continent from world group by continent having sum(population)>100000000;

十一、JOIN

1. 什么是join
	合拼两个或多个表格
	语句FROM 表示合拼两个表格game 和 goal的数据 ，语句 ON 表示如何找出 game中每一列应该配对goal中的哪一列
2. 代码演示
	select team1,team2,player from game join goal on (id=matchid) where player like 'Mario%';
3. 补充知识
	上述代码又可以写成 
		select team1,team2,player from game,goal where id=matchid and player like 'Mario%';
	如果是合拼三个表格
		select name from movie,actor,casting where movie.id = movieid and actor.id = actorid and title = 'Alien';
	三个表格 jion 写法
		select name from movie join casting on movieid = movie.id join actor on actorid = actor.id where title = 'Alien'；
