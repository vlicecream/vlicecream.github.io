   ```bash
   sudo steamos-readonly disable
   ```

## ***RimWorld***

### ***必装mod***

1. *HugsLib*
   - *主要功能是为其他mod提供共享功能，可以简单理解为其他mod能够正常使用的基础，本身在游戏中没有功能体现，但几乎所有mod都需要该mod作为前置*
2. *Harmony*
   - *主要功能是为其他mod提供共享功能，可以简单理解为其他mod能够正常使用的基础，本身在游戏中没有功能体现，但几乎所有mod都需要该mod作为前置*
3. *Humanoid Alien Races*
   - *一切新物种mod的添加都需要该mod支持*

### ***优化原功能mod***

1. *Prepare Landing*
   - *Prepare Landing - 简&繁中文汉化包*
   - *非常庞大详细的内容筛选器，允许玩家在降落前筛选殖民地的气候、温度、土壤等条件*
2. *RandomPlus*
   - *RandomPlus_zh*
   - *可随机或者定制小人的各项属性，包括技能、癖好、旧伤以及成瘾品上瘾等*
3. *Some more jobs*
   - *Some more jobs_繁简补充，并没有完全汉化，但是足够用了*
   - *‌细化了原版简单粗暴的分工方式，将原版诸如医生的职务包括了照顾病人、医疗病人、为病人做手术等专业与非专业工作，与给酿酒桶补充麦芽和填充弹药等明显不同类型的分工区分开来。节省人力的同时，提高了效率，堪称强迫症福音*
4. *Pick Up And Haul*
   - *pick up and haul zh 中文补丁*
   - *改变了游戏原版小人一次只能拿一样物品的设定，取而代之的是只要重量允许，小人将一次拿取多种不同类型的物品*
   - *很方便 搬运效率杠杠的*
5. *Automatic Night Owl*
   - *无需汉化*
   - *基于游戏中具有夜猫子特质的小人在白天工作效率很低，夜晚工作效率翻倍的设定，该mod会自动将具有这类特质的小人的作息表安排成白天休息，晚上工作*
6. *Defensive Positions*
   - *看了下没有找到汉化mod*
   - *允许你的小人记住自己的防御位置，需要时，可一键指派各位小人前往自己防御位置，此外玩家可快速创建、快速选择小人，并且支持基础创建模式和高级创建模式(一个小人可最多记住四个防御位置)*
7. *Auto-Cut Blight*
   - *无需汉化*
   - *枯萎病会使感染的作物不再生产，并且感染性极强，一株感染，不过半天，整片作物田都难幸免，是大规模种植的噩梦*
   - *本mod会在枯萎病发生时，自动指派具有种植工作的小人前往铲除病株，将损失降到最小*
8. *Beautiful Bodies*
   - *无需汉化*
   - *顾名思义，将本体不论男女都是棍子身材的外形进行美化，至于效果嘛，不便多说，你懂的*
9. *Beautiful Outdoors*
   - *无需汉化*
   - *对室外自然景色进行了美化，并且增加了小人对不同景物的情绪反馈，简言之，美丽的景观加心情，丑陋的景观减心情，并且相比原版，设定更完善*
10. *[NL] Facial Animation - WIP*
    - *增加了小人的面部动画，现在小人会眨眼、皱眉、微笑、生气等情绪化表达*
11. *Camera+*
    - *功能十分强大的mod，大大增加了可放大/可缩小倍率的同时，丝毫不减少画面的精美度，现在你甚至能放大到单个小人观察他的脸部活动，或者放大到整个地图观察某些原本不可察的变化*
12. *Wall Light*
    - *Wall Light -- 壁灯 简体汉化包*
    - *‌新增了一个无需科技解锁的壁灯，相比原版初始给的照明物品火把，简单易用而高效*
13. *Hospitality*
    - *Hospitality - 简&繁中文汉化包*
    - *增加了拜访小人的活动，现在的友方阵营小人拜访玩家基地时不再像以前一样只是到处乱逛，而是能够与之交流增进感情，并且可以为游客安排住宿和娱乐等活动(当然不是免费滴)*
    - *严格意义来说，这个mod其实在一定程度上改变了游戏的玩法。但是在我看来，这个mod设计的功能是原版就应该要包括的，但是泰南偷懒所以没搞，因此我把这个mod也选取进来了，是否使用，还是看你的需要*
14. *Hospitality: Casino*
    - *Hospitality: Casino - 简&繁中文汉化包*
    - *新增了老虎机娱乐设备，你的小人可以免费游玩，但是访客必须付钱来作为赌注*
    - *需要 前置mod - Hospitality*
15. *Colony Manager*
    - *Colony Manager - 简繁中文汉化包*
    - *为小人以及动物增加了更多的自动化工作设定*
    - *例如当肉量低于设定的数量时，小人会自动出去狩猎*
      - *比如自动清理风力涡轮发电机的集水区，使发电机时刻保持最高效率*
      - *比如为动物设置不同的年龄、性别组合分类，设置自动分配小人屠宰、驯服与训练等*
      - *合理利用的话，既能提高效率，又十分省心*
