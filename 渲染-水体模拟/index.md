# 渲染 - 水体模拟


## ***波动方程***

$$
F(x) = A \cdot \sin(\frac{2\pi}{T}x + p)
$$

### ***Unreal材质实现***

*这个材质实现了一个基于世界坐标x轴的简单正弦顶点位移。简单来说他能让物体像海浪一下上下波动*

*参数介绍：*

| *参数名*         | *对应公式* | *作用*                                                       |
| ---------------- | ---------- | ------------------------------------------------------------ |
| ***WaveLength*** | *T*        | *波长。数值越大，波越平缓（宽）；数值越小，波越密集（窄）。* |
| ***Amplitude***  | *p*        | *振幅。数值越大，波浪起伏越高；设为 0 则变回平面。*          |
| ***Phase***      | *A*        | *相位。改变它会让波浪产生位移，通常用来做动画。*             |
